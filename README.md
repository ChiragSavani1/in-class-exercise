# In-Class Exercise: Unit Testing with Jest

## 🚀 Setup Instructions

1. Clone this repository: 
git clone <repo-link> cd in-class-exercise

2. Install Jest:
npm install

3. Run the tests:
npm test

## 🏆 Your Task
Complete the missing test cases in `index.test.js`. Run `npm test` to verify your tests.
